== Description ==

"A Te" is a slight modification of the "Hello, Dolly" plugin that comes with Wordpress. When activated, it displays the lyrics to the Lorenzo Cherubini song "A Te" in the top right corner of the admin panel.

== Installation ==

1. Upload `a te.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Fall in love every time you write a new post

== Frequently Asked Questions ==

Q. does it work with "Hello, Dolly"? 

A. they both work, but the lyrics overlap each other. If you really want them both at once, edit your "a te.php" file and change the line "top: 2.3em;" to "top: 3.3em".
